# 🎮 Ultimate Quiz Game

An interactive quiz web application built using HTML, CSS, and JavaScript.  
Users can select different categories and test their knowledge in a fun way.

---

##Features
- Multiple categories (Science, Movies, Cartoons, Sports)
- Interactive UI with modern design
- Quiz-based question system
- Start game functionality
- Responsive design

---

## Technologies Used
- HTML
- CSS
- JavaScript
